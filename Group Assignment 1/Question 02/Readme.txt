To run the test cases just run "solution.py" on python 2.7
To run your own test case :
1)create the input tree structure using "Tree" class
2)Insert Leaf nodes with method "Tree.insertNode('name', 'rating', 'parent name')"
3)For the root node use "Tree.insertNode('name', 'rating')" method to insert root node to the tree
4)Run method "solve(Tree)" to get the solution as a python list(list containg initee names)
5)to stop running the default test case comment line 249 in solution.py
